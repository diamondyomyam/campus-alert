# เว็บประชาสัมพันธ์โรงเรียนบ้านเป้าวิทยา (Real-Time Board)

โครงสร้างไฟล์ แยกเป็น HTML / CSS / JS เพื่อให้ศึกษาและแก้ไขง่ายใน VS Code

```
school-board/
├── index.html   → โครงหน้าเว็บ (เปิดไฟล์นี้ในเบราว์เซอร์เพื่อดูผลลัพธ์)
├── style.css    → สีธีม ฟ้า-แสด, ดีไซน์การ์ด, ป้าย LED, นาฬิกาฟลิป, ปุ่มแจ้งเตือน
└── script.js    → โค้ดการทำงานทั้งหมด
```

## วิธีเปิดดู
เปิดไฟล์ `index.html` ด้วยเบราว์เซอร์ได้เลย (ดับเบิลคลิก หรือใน VS Code คลิกขวา → "Open with Live Server" ถ้าติดตั้งส่วนขยาย Live Server ไว้)

## จุดที่น่าศึกษาใน script.js (เรียงตามลำดับในไฟล์)

| ส่วน | ทำอะไร | ตัวแปร/ฟังก์ชันหลัก |
|---|---|---|
| นาฬิกาฟลิป | อัปเดตเวลาจริงทุกวินาที พร้อมวันที่ไทย พ.ศ. | `updateClock()`, `setDigit()` |
| ป้าย LED ข่าวด่วน | วิ่งข้อความสลับทุก 9 วิ | `ledItems`, `cycleLed()` |
| ปฏิทินกิจกรรม | ลิสต์กิจกรรม กรองได้ตามคำค้นหา | `agenda`, `renderAgenda()` |
| ข่าวประชาสัมพันธ์ | ฟีดข่าว นับเวลา "...นาทีที่แล้ว" | `newsData`, `renderNews()` |
| บอร์ดประกาศ | โพสต์อิทสีตามความเร่งด่วน | `board`, `renderBoard()` |
| ระบบค้นหา | กรอง+ไฮไลต์คำค้นหาทั้ง 3 ส่วนพร้อมกัน | `currentSearch`, `renderAll()`, `hl()` |
| ระบบแจ้งเตือน | กระดิ่ง + toast popup เมื่อมีอัปเดตใหม่ | `notifications`, `pushNotification()`, `showToast()` |
| ข้อมูลจำลองอัปเดตสด | รายการข่าว/ประกาศที่จะ "เด้ง" เข้ามาทีละรายการ | `livePool` |

## แก้ไขที่ใช้บ่อย

- **เปลี่ยนชื่อ/ที่อยู่โรงเรียน** → แก้ใน `index.html` (หัวเว็บกับ footer)
- **แก้/เพิ่มกิจกรรมในปฏิทิน** → แก้อาเรย์ `agenda` ใน `script.js`
- **แก้/เพิ่มข่าวประชาสัมพันธ์** → แก้อาเรย์ `newsData`
- **แก้/เพิ่มประกาศบอร์ด** → แก้อาเรย์ `board`
- **แก้ข้อความวิ่งบนป้าย LED** → แก้อาเรย์ `ledItems`
- **ปรับความถี่การแจ้งเตือนสด** → แก้เลข `45000` (มิลลิวินาที) ที่ท้ายไฟล์ `script.js`
- **ปรับโทนสี** → แก้ค่าตัวแปรสีในส่วน `:root{ ... }` บนสุดของ `style.css`

## หมายเหตุ
ข้อมูลทั้งหมด (ข่าว/กิจกรรม/ประกาศ) เป็นข้อมูลจำลองอยู่ในตัวแปร JavaScript เท่านั้น ยังไม่เชื่อมต่อกับฐานข้อมูลหรือเซิร์ฟเวอร์จริง เหมาะสำหรับสาธิตแนวคิด UI/UX ของระบบเรียลไทม์ในโครงงาน
